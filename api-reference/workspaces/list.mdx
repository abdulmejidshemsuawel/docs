## List Workspaces

Retrieves a list of all workspaces the authenticated user is a member of.

### Request

<ParamField header="Authorization" required>
  Bearer token for authentication
</ParamField>

### Response

<ResponseField name="data" type="Array<Workspace>">
  Array of Workspace objects
</ResponseField>

### Error Codes

<ResponseField name="401" type="Unauthorized">
  Missing or invalid API token
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Token does not have 'workspaces' permission
</ResponseField>
