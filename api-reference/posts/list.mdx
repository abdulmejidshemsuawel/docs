## List Posts

Retrieves a paginated list of all your posts across your active workspaces. Posts are sorted by creation date in descending order.

### Request

<ParamField header="Authorization" required>
  Bearer token for authentication
</ParamField>

### Response

<ResponseField name="data" type="Array<Post>">
  Array of Post objects
</ResponseField>

<ResponseField name="links" type="Object">
  Pagination links
</ResponseField>

<ResponseField name="meta" type="Object">
  Pagination metadata
</ResponseField>
