## List Labels

Retrieves a list of all labels across all workspaces the authenticated user has access to.

### Request

<ParamField header="Authorization" required>
  Bearer token for authentication
</ParamField>

### Response

<ResponseField name="data" type="Array<Label>">
  Array of Label objects
</ResponseField>

### Error Codes

<ResponseField name="401" type="Unauthorized">
  Missing or invalid API token
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Token does not have 'labels' permission
</ResponseField>
