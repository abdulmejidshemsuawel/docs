## Update Label

Updates the details of an existing label.

### Request

<ParamField header="Authorization" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" required>
  application/json
</ParamField>

<ParamField path="label_id" type="integer" required>
  The unique identifier of the label to update
</ParamField>

<ParamField body="name" type="string">
  The updated name of the label (max 50 characters)
</ParamField>

<ParamField body="color" type="string">
  The updated hex color code for visual identification of the label
</ParamField>

<RequestExample>
```json
{
  "name": "Special Promotion",
  "color": "#8e24aa"
}
```
</RequestExample>

### Response

<ResponseField name="id" type="integer">
  Unique identifier of the label
</ResponseField>

<ResponseField name="name" type="string">
  The updated name of the label
</ResponseField>

<ResponseField name="color" type="string">
  The updated hex color code of the label
</ResponseField>

<ResponseField name="workspace" type="object">
  Information about the label's workspace
</ResponseField>

<ResponseField name="postsCount" type="integer">
  The number of posts associated with this label
</ResponseField>

<ResponseField name="createdAt" type="string">
  Creation timestamp (ISO 8601)
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Last update timestamp (ISO 8601)
</ResponseField>

<ResponseExample>
```json
{
  "id": 4,
  "name": "Special Promotion",
  "color": "#8e24aa",
  "workspace": {
    "id": 1,
    "name": "Marketing Team"
  },
  "postsCount": 5,
  "createdAt": "2023-11-22T11:30:00Z",
  "updatedAt": "2023-11-25T09:45:00Z"
}
```
</ResponseExample>

### Error Codes

<ResponseField name="400" type="Bad Request">
  Validation errors or invalid parameters
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Missing or invalid API token
</ResponseField>

<ResponseField name="403" type="Forbidden">
  Token does not have 'labels' permission
</ResponseField>

<ResponseField name="404" type="Not Found">
  Label not found or user doesn't have access to the label's workspace
</ResponseField>
