## Media Optimization

PostSyncer automatically optimizes your media for each platform's requirements:

<AccordionGroup>
  <Accordion icon="crop" title="Image Sizes">
    We automatically crop and resize your images to the recommended dimensions for each platform. For example:
    
    | Platform | Recommended Aspect Ratio | Max File Size |
    |----------|--------------------------|---------------|
    | Instagram | 1:1, 4:5, 16:9 | 8MB |
    | Twitter | 16:9 | 5MB |
    | Facebook | 1.91:1 | 10MB |
    | LinkedIn | 1.91:1 | 5MB |
  </Accordion>
  <Accordion icon="video" title="Video Formats">
    We handle video compression and format requirements for each network:
    
    | Platform | Optimal Length | Max File Size |
    |----------|---------------|---------------|
    | TikTok | 15-60 seconds | 4GB |
    | Instagram | 3-60 seconds | 100MB |
    | Twitter | 140 seconds | 512MB |
    | YouTube | Any | 10GB |
  </Accordion>
</AccordionGroup>
